RuckR Agent Information

RuckR is a rugby-themed, location-based creature collector. Players explore the real world, discover rugby pitches, collect rugby player creatures, and battle other players.

Authentication Boundaries

Most gameplay pages and API routes require an authenticated player session. Public discovery should use this page, the home URL, the sitemap, and llms.txt.

Sensitive Actions

GPS, recruitment, pitch creation, battle, profile, avatar, privacy, and account actions are user-sensitive. Agents should not perform those actions unless the user explicitly asks and authentication is available.

Agent Action API

RuckR Agent Discoverability V1 does not expose an OpenAPI document, MCP server, or other agent action API. This page is informational only.
