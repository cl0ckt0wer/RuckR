# Recall Memory

Search agentmemory for context from past sessions.

**Use the `memory_smart_search` tool** with the provided query to find:
- Past architectural decisions
- Bug fixes and their root causes
- Recurring patterns and conventions
- Files frequently modified for similar tasks

If no query is provided, ask the user what they're working on and search for relevant context.

Before making any code changes, check memory for:
1. Related past bugs
2. Files that were modified together
3. Patterns/conventions the project follows
4. Lessons learned from similar work
